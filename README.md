## Installation

### Requirements

- [NodeJS](https://nodejs.org/en/download)
- Xampp

### Process

- run your xampp server

```bash
git clone https://github.com/dimaliwatkent/budget-tracker.git
cd budget-tracker
npm install

=== to run the project ===
npm start
```
